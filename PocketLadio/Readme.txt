PocketLadio


【主な機能】
PocketLadioはMicrosoft Windows Mobile 5.0 Pocket PC上で動作する「ねとらじ」の
ヘッドラインをインターネットより取得、表示するツールです。

ねとらじ
http://live.ladio.livedoor.com/


【必要環境】
Microsoft Windows Mobile 5.0 Pocket PC
.NET Compact Framework 1.0 SP3

.NET Compact Frameworkについては
http://www.microsoft.com/japan/msdn/vstudio/device/netcf/
を参照してください。


【インストール】
ZIPファイルを解凍し、PocketLadio.exeを適当なフォルダに置いてください。


【アンインストール】
PocketLadio.exeと、同じフォルダにあるSetting.xmlおよびSetting.*.xmlを削除して
ください。
PocketLadioはレジストリを使用しません。


【使い方】
・PocketLadioの使い方
Getボタンでねとらじで配信中のヘッドラインを取得します。
Playボタンで選択した番組を再生します。
Filterチェックボックスであらかじめ設定したフィルタ条件に一致する番組のみを表示
します。

・PocketLadioの設定
トップ画面から[メニュー]->[設定]->[PocketLadio設定]を選択するとPocketLadioの設
定を行うことができます。
[ヘッドラインの自動チェック間隔]はヘッドラインを自動でチェックする場合の、
チェック間隔を設定することができます。
[メディアプレーヤーのパス]は番組を再生するためのメディアプレーヤーのファイルパ
スを設定します。
メディアプレーヤーにはTCPMP（http://tcpmp.corecodec.org/about）をおすすめしま
す。
[ブラウザのパス]は番組のウェブサイトにアクセスするためのブラウザのファイルパス
を設定します。

・ねとらじ設定
トップ画面から[メニュー]->[設定]->[ねとらじ設定]を選択するとねとらじの設定を行
うことができます。
[ヘッドラインのURL CVS]・[ヘッドラインのURL XML]はねとらじのヘッドラインのURL
を指定します。
通常は初期状態のままで構いません。
[ヘッドラインの取得方法]はねとらじのヘッドラインの取得方法を設定します。
通常はCVSをおすすめします。（ねとらじのXMLは不正なXMLを出力していることが多い
ため）
[ヘッドラインの表示方法]はトップ画面でのヘッドラインの表示方法を設定します。

・ねとらじ設定 ヘッドラインの表示方法
[[NAME]]  - 番組のタイトル
[[GENRE]] - 番組のジャンル
[[CLN]]   - 現リスナ数
[[CLNS]]  - 延べリスナ数
[[TITLE]] - 現在の曲名情報
[[TIMES]] - 放送開始時間
[[BIT]]   - ビットレート
